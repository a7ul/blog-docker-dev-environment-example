# Sample Go service

This is a super simple example go lang based service.

## To run for development

### For Windows

```shell
SET PORT=5000
go run .
```

or to build the application :

```shell
go build
SET PORT=5000
go1.exe
```

### For Mac, Linux or Windows WSL

```shell
export PORT=5000
go run
```

or

```shell
PORT=5000 go run .
```

or

```shell
go build
PORT=5000 ./go1
```
